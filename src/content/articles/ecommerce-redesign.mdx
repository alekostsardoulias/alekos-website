# E-Commerce Platform Redesign

A full-stack rebuild of a legacy e-commerce platform serving 10,000+ monthly active users. The project involved migrating from a monolithic PHP application to a modern, serverless architecture.

## What We Built

The platform runs on Next.js with the App Router, using Supabase for the database and authentication, Stripe for payment processing, and Vercel for deployment. The admin dashboard provides real-time inventory management, order tracking, and customer analytics.

## Key Results

- Page load times reduced by 60% (from 3.2s to 1.3s LCP)
- Conversion rate improved by 18% after checkout flow redesign
- Zero-downtime migration from the legacy system
- Admin team reported 40% faster daily operations
